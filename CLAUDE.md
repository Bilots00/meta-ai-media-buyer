# CLAUDE.md

**Hosting / deploy: Railway — NOT Manus.** Despite the Manus template origins visible in `README.md` and `template.json`, this app is deployed on **Railway**: pushing to `main` triggers a Railway deploy, and all environment variables live in the **Railway dashboard** (Variables tab). Manus is at most a provider of some backend services via env vars — the site does not run on Manus.

👉 See **[AGENTS.md](AGENTS.md)** for the full, authoritative project facts (stack, structure, routing, POD Partners, deploy flow).
